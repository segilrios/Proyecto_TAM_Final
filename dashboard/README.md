# Dashboard

Archivo principal:

```text
dashboard_interactivo_final.html
```

Es un dashboard autocontenido con:

- selector de modelo,
- resultados de clasificación,
- regímenes ABS/CHG,
- explicabilidad,
- auditoría,
- simulador de estrategias.

Para abrirlo con servidor local:

```bash
python scripts/04_run_dashboard_server.py
```
