# Depuración del repositorio

Esta versión fue limpiada para presentación y GitHub.

## Decisiones tomadas

- Se eliminó la colección de notebooks históricos.
- Se dejó un único notebook final: `notebooks/proyecto_oro_colombia_final.ipynb`.
- Se eliminaron ZIPs internos duplicados.
- Los modelos se guardan solo en `models/`.
- Las figuras se guardan solo en `outputs/figures/`.
- Los datos fuente/proxy están en `data/raw/`.
- Los resultados procesados están en `data/processed/`.
- El dashboard final queda en `dashboard/dashboard_interactivo_final.html`.
- El código ejecutable queda en `scripts/`.

## Versiones antiguas

Las versiones intermedias no se incluyen para evitar ruido visual y confusión al presentar el proyecto.
